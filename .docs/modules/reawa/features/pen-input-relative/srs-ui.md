# SRS — Relative Pen Input (UI)

## [SRS-RW-33] Relative mode selection

- Settings: segmented output mode **Relative** (default).
- Menu bar: **Relative** item with green dot when active; greyed when already Relative.
- Switching from Absolute while connected applies immediately via live config ([SRS-RW-12](../connection-management/srs-logic.md)).

No on-screen indicator during normal use beyond menu bar state.
