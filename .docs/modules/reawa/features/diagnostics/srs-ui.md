# SRS — Diagnostics (UI)

## [SRS-RW-65] Settings log tabs

Settings window tabbed surface includes:

| Tab | Default | Behavior |
|---|---|---|
| **App Behavior Log** | Always on | Searchable stream of app-level events |
| **Pen Event Log** | Off | **Capture pen events** toggle enables high-frequency stream |

Both tabs support search/filter. Pen tab capability chips fill search on click.

Connections tab is separate (connection-management feature).
